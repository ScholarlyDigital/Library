IMPORTANT NOTICE

Scholarly is Changing

Scholarly is rebranding and establishing itself as Scholarly Digital, and this change is a long time coming that will lead the way to a revamp and rebrand of all Scholarly services. Many of the existing projects are too scrappy and outdated, Library being one of them. Detailed information can be found on our GitHub (and our website in the near future), but for now we will list what will happen to the big services in the near future.

Library

Intially Library was an experiment to introduce people to Coach, as he was an early development product meant for the full release of Scholarly. Since then, the direction of the project changed dramatically, and what was 'The Scholarly' has now been cancelled. Coach however, has recieved many advancements since release, increasing speed, intelligence, memory, and most recently, seeing images. Coach is planned to live (discussed later), but Library will be shutdown. Coach will need a platform to be interfaced with due to the existing user base, so a stable version of Library will be shifted here and will be labelled as Legacy. When the initial Library site shuts down in December 2023, Legacy will take it's place until March 2024, where the Legacy site will too be shut down. The Legacy site is on life support, meaning it will recieve bug fixes, but no new feautures will be added.

Coach

Speaking of Coach, he is also due for a revamp. The additional features he recieved over time has racked up some serious technical debt, and he is due for a rework from the ground up. Planned features for the new Coach will include voice input, text-to-speech, image generation, accounts, premium and free tiers and way more. This remake will need a new client, which means the new version of Coach will recieve his own web application in early 2024, so keep an eye out for that.

API

Lastly, the API. Just like Coach, the API is due for a remake with a ton more freedom for developers. The v1 API is scheduled to continue running until March 2024, and will be relabled to Legacy when the new API is released. The new API will require developer accounts and tokens, in return for more flexibility and features (that expand past AI and Coach). All new Coach features mentioned will be accessible via the API. As Coach is built on a custom branch of the API (which is a seperate group of endpoints with slightly modified code for independnce), developers can also request their own branches for projects. An early version of the API rework is scheduled to release in December 2023, but the stable version (and the documentation) will release and replace the old API sometime in early-to-mid 2024.
That being said we hope you stick around with us for this new chapter in Scholarly's history and see the many incredible things we have planned for the education industry as a whole.
- The Scholarly Digital Team
